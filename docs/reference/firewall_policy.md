# Firewall Policy

Create firewall policies.


## Versions
- Ansible v2.10.4
- Ansible collection fortinet.fortios v2.0.0
- FortiOS 6.4


## Usage


=== "CLI"

    ```bash
    --8<-- "code/fortios/config_firewall_policy.conf"
    ```

=== "Ansible"

    ```yaml
    --8<-- "code/ansible/fortios_firewall_policy.yml"
    ```

=== "Terraform"

    ```terraform
    --8<-- "code/terraform/fortios_firewall_policy.tf"
    ```
